# Employee Attrition Analytics System

## Overview
Employee Attrition Analytics System developed using Python, MySQL, Pandas and Matplotlib.

## Features
- Imported 4653 employee records into MySQL
- SQL-based analytics
- Attrition Dashboard
- City-wise analysis
- Education-wise analysis
- Gender-wise analysis
- Payment Tier analysis
- Experience-based attrition analysis

## Technologies Used
- Python
- MySQL
- Pandas
- Matplotlib
- SQL
